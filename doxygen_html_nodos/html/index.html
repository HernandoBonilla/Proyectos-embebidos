Documentación Doxygen - Sistema Distribuido de Dos Nodos

Nodo 1 Nodo 2 Arquitectura Doxyfile mainpage.md

Este paquete contiene una versión lista para usar de la documentación del sistema en estilo Doxygen para dos nodos:

  • Nodo 1: ESP32-S3 principal con LCD, RTC, pulso e IMU local.
  • Nodo 2: ESP32 portátil con MPU6050 en pecho.

Cómo generar HTML real con Doxygen:

  1. Copia Doxyfile, mainpage.md y los bloques Doxygen a tu proyecto.
  2. Instala Doxygen.
  3. Ejecuta doxygen Doxyfile.
  4. Abre docs/html/index.html.
